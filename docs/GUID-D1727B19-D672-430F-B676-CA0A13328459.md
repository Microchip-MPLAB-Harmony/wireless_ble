# Extended scan data length

## C

```c
#define BLE_GAP_EXT_SCAN_DATA_LEN_MIN                           0x004A
#define BLE_GAP_EXT_SCAN_DATA_LEN_MAX                           0x0672
```

## Description

Range of extended scan data length.

## Macro Definition

|Macro|Description|
|:---|:---|
|BLE_GAP_EXT_SCAN_DATA_LEN_MIN|Minimum extended scan data length.|
|BLE_GAP_EXT_SCAN_DATA_LEN_MAX|Maximum extended scan data length.|
