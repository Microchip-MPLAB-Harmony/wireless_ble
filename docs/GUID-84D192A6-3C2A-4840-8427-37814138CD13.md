
## Description

Handle BLE\_DD \(Database Discovery middleware\) events.<br />This API should be called in the application while caching BLE\_DD events.

## Parameters

|Parameter|Description|
|---------|-----------|
|\[in\] p\_event|Pointer to BLE\_DD events buffer.|
