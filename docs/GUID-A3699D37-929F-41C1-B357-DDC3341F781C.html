BT_SYS_AllowSystemSleep

C

bool BT_SYS_AllowSystemSleep(uint32_t ulRtcFrequency, uint32_t ulRtcCurrentCnt);

Description

Query if BT allow system enter sleep mode.

Parameters

ParameterDescription
[in] ulRtcFrequencyRTC frequency.
[in] ulRtcCurrentCntRTC current count.

Return values

Return valueDescription
trueBT is sleeping and BT is ready for System enter sleep.
falseBT is not sleeping or BT is not ready for System enter sleep.
