GATT Sub-Procedure Requirements

Requirements in this section represent a minimum set of requirements for an OTA Client. Other GATT sub-procedures may be used if supported by both OTA Client and Server. The table below summarizes additional GATT sub-procedure requirements beyond those required by all GATT clients.

Discover All Characteristics Descriptors M

Read Characteristic Descriptors M
