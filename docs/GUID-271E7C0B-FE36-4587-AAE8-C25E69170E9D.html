2.2.4.5.2 Uplink Credit Based Flow Control

The procedure is used by a Transparent Client to enable Uplink Credit Based Flow Control on the Transparent Server. Once it is enabled, Server role can only send data to Client while the credit count is greater than zero. The detail information is described in Transparent Service [1].
