# Maximium build-in CCCD number

## C

```c
#define GATTS_BUILDIN_CCCD_NUM                              0x01
```

## Description

The definition of total CCCD number of build-in services.

## Macro Definition

|Macro|Description|
|:---|:---|
|GATTS_BUILDIN_CCCD_NUM|There exist one CCCD in build-in GATT Services.|
