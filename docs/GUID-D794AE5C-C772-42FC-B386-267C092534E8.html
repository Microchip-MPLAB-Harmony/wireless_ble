BLE_HOGPS_SetBootKeyboardInputReport

C

uint16_t BLE_HOGPS_SetBootKeyboardInputReport(uint8_t *p_keyCodeArray);

Description

Set HID boot keyboard input report.

Parameters

ParameterDescription
[in] p_keyCodeArray8 bytes of key codes array. From byte 0 to byte 7: |Modifier Keys|Reserved|Keycode 1|Keycode 2|Keycode 3|Keycode 4|Keycode 5|Keycode 6|.

Return values

Return valueDescription
MBA_RES_SUCCESSSuccessfully set the input report.
MBA_RES_INVALID_PARAInvalid parameters.
