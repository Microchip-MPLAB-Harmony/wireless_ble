```c
typedef bool (*BT_SYS_PtaGetWlanStatusCb_T)(void);
```

## Description

PTA callback type. This callback function get PTA wlan\_active status from the application.

## Return values

|Parameter|Description|
|---------|-----------|
|true|WLAN uses RF.|
|false|BLE can use RF.|
