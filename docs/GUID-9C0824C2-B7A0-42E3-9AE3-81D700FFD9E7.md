## C

```c
uint16_t BLE_ANCS_Init();
```

## Description

Initialize ANCS profile.

## Return values

|Return value|Description|
|------------|-----------|
|MBA\_RES\_SUCCESS|Initialize BLE ANCS profile successfully.|
|MBA\_RES\_FAIL|Fail to initialize BLE ANCS profile.|
