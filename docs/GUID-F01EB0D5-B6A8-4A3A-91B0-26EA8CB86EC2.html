BLE_PXPR_EventId_T

C

typedef enum BLE_PXPR_EventId_T
{
    BLE_PXPR_EVT_LLS_ALERT_LEVEL_WRITE_IND = 0x00,
    BLE_PXPR_EVT_IAS_ALERT_LEVEL_WRITE_IND,
    BLE_PXPR_EVT_ERR_UNSPECIFIED_IND
}BLE_PXPR_EventId_T;

Description

Enumeration type of BLE PXP Reporter callback events.

EnumeratorDescription
BLE_PXPR_EVT_LLS_ALERT_LEVEL_WRITE_INDEvent for Alert Level in Link Loss Service is written. See BLE_PXPR_EvtAlertLevelWriteInd_T for event details.
BLE_PXPR_EVT_IAS_ALERT_LEVEL_WRITE_INDEvent for Alert Level in Immediate Alert service is written. See BLE_PXPR_EvtAlertLevelWriteInd_T for event details.
BLE_PXPR_EVT_ERR_UNSPECIFIED_INDProfile internal unspecified error occurs.
