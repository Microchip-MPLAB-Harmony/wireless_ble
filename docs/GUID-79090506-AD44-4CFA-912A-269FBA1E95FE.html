Functions

  • MW_DFU_Config

    Configures the device information for the DFU process.

  • MW_DFU_FwImageActivate

    Activates the new firmware after a system reboot.

  • MW_DFU_FwImageRead

    Reads a fragment of the firmware image from flash.

  • MW_DFU_FwImageStart

    Starts or restarts the firmware image update procedure.

  • MW_DFU_FwImageUpdate

    Updates a fragment of the firmware image to flash.

  • MW_DFU_FwImageValidate

    Validates the updated MCU image using CRC for internal flash or checksum for external flash.
