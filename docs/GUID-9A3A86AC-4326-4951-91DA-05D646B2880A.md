# SPSM range

## C

```c
#define BLE_L2CAP_SPSM_DYNAMIC_MIN                  0x0080
#define BLE_L2CAP_SPSM_DYNAMIC_MAX                  0x00ff
```

## Description

Range for SPSM.

## Macro Definition

|Macro|Description|
|:---|:---|
|BLE_L2CAP_SPSM_DYNAMIC_MIN|The minimum dynamic SPSM value|
|BLE_L2CAP_SPSM_DYNAMIC_MAX|The maximum dynamic SPSM value|
